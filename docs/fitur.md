# Fitur Aplikasi Novos

## CUSTOMER
- Register & Login
- Lihat katalog produk jersey
- Buat pesanan custom jersey (isi form desain)
- Upload logo/desain
- Konfirmasi pesanan (ACC setelah dicek admin)
- Tracking status pesanan
- Riwayat pesanan
- Pembayaran via Midtrans
- Chat dengan admin (terikat per pesanan)

## ADMIN
- Login
- Dashboard ringkasan pesanan & pembayaran
- Kelola pesanan masuk
- Cek & konfirmasi pesanan ke customer
- Teruskan pesanan ke tim Design
- Kelola pengguna (customer & pegawai)
- Kelola pembayaran
- Kelola chat customer
- Laporan pesanan

## DESIGN
- Login
- Lihat pesanan yang masuk ke divisi design
- Print detail pesanan & desain
- Tandai selesai → teruskan ke Produksi

## PRODUKSI
- Login
- Lihat tugas produksi yang masuk
- Update status tugas (dikerjakan / selesai)

## SUPER ADMIN / MANAGER
- Semua akses Admin
- Kelola role & pegawai
- Lihat laporan lengkap
- Akses fitur Stress Test pegawai

## FITUR INTERNAL — STRESS TEST
- Khusus pegawai (bukan customer)
- Mengisi kuesioner stress test
- Melihat hasil/skor stress test pribadi
- Admin/Manager bisa lihat semua hasil
- Pertanyaan bisa dikelola (tambah/edit/hapus) oleh Super Admin
